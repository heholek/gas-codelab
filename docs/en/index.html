In this codelab, You can make a application that exports Gmail messages to Google Spreadsheet.

You need own Gmail or G Suite account.

If you use Gmail for the first time, you need to do below

1. Open mail-to-me application

Open the application in your browser.
mail-to-me

2. Make a copy

Select File > Make a Copy.

3. Run mail-to-me

Select Run button.

4. Confirm your mail

Open Gmail in your browser and confirm to receiving emails.

First, create a Apps Script project. Visit script.google.com to open the script editor. Click + New Project to proceed to the script editor. Click Untitled Project and input your project name. Then it appears toast and it complete to save when the toast disappears.

Next, understand GmailApp class. Confirm Google apps classes in official reference. See GmailApp document. You can confirm GmailApp classes (ex. GmailMessage, GmailThread ) and methods (ex. search, sendEmail)

we use getInboxThreads method for Retrieving emails in this time.
